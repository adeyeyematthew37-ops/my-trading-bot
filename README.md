# 🤖 CryptoBot — Multi-Chain Telegram Trading Bot

A fully-featured Telegram trading bot supporting **7 blockchains**, with built-in wallet generation, automated DCA bots, token swaps, and price alerts.

## ✨ Features

| Feature | Description |
|--------|-------------|
| 👛 **Wallet Generator** | Generate/import wallets for all chains |
| 💱 **Token Swaps** | Buy, sell & swap via DEX aggregators |
| 🤖 **DCA Bots** | Automated dollar-cost averaging |
| 🔔 **Price Alerts** | Get notified at target prices |
| 💰 **Price Feeds** | Real-time prices via CoinGecko & DexScreener |
| 📜 **Trade History** | Full history of all transactions |

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- A Telegram account
- Free API keys (see below)

### 1. Clone & Install
```bash
git clone https://github.com/adeyeyematthew37-ops/my-trading-bot.git
cd my-trading-bot
npm install
```

### 2. Configure
```bash
# Run the interactive setup wizard
npm run setup

# OR manually copy and edit
cp .env.example .env
nano .env
```

### 3. Start
```bash
npm start

# Development mode (auto-restart)
npm run dev
```

## 📖 Bot Commands

### 👛 Wallet Commands
```
/wallet              - View all your wallets
/generate [chain]    - Generate a new wallet
/import [chain] [key] - Import from private key
/balance [chain]     - Check wallet balances
/send [chain] [to] [amount] - Send tokens
/export [chain]      - Export private key (auto-deletes in 30s)
```

### 💱 Trading Commands
```
/buy [chain] [token] [amount]           - Buy a token with native currency
/sell [chain] [token] [amount]          - Sell tokens for native currency
/swap [chain] [tokenIn] [tokenOut] [amount] - Swap any two tokens
/history                                - View recent trades
```

### 🤖 DCA Bot Commands
```
/dca                 - DCA overview & menu
/newdca [chain] [tokenIn] [tokenOut] [amount] [freqMinutes] [totalOrders?]
/dcalist             - View all DCA orders
/stopdca [id]        - Cancel a DCA order
```

### 💰 Price Commands
```
/price               - Show all native token prices
/price [symbol]      - Get price by name/symbol
/price [chain] [address] - Get price by contract address
/search [query]      - Search for tokens
/chart [token]       - View chart links
```

### 🔔 Alert Commands
```
/alert [chain] [address] [above|below] [price] - Set price alert
/alerts              - View active alerts
/stopalert [id]      - Remove an alert
```

## 💡 Usage Examples

### Generate a Wallet
```
/generate ethereum
/generate solana
/generate bsc
```

### Buy Tokens
```
# Buy USDC with 0.01 ETH on Ethereum
/buy ethereum 0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48 0.01

# Buy CAKE with 0.05 BNB on BSC
/buy bsc 0x0E09FaBB73Bd3Ade0a17ECC321fD13a19e81cE82 0.05
```

### DCA Examples
```
# Buy USDC daily with 0.01 ETH (unlimited)
/newdca ethereum native 0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48 0.01 1440

# Buy CAKE every hour, 48 times total
/newdca bsc native 0x0E09FaBB73Bd3Ade0a17ECC321fD13a19e81cE82 0.05 60 48

# Buy SOL-based token weekly
/newdca solana native TokenMintAddress 0.1 10080
```

### Price Alerts
```
# Alert when ETH goes above $5000
/alert ethereum native above 5000

# Alert when a token drops below $0.001
/alert ethereum 0xTokenAddress below 0.001
```

## 🔐 Security

- Private keys are encrypted with **AES-256-GCM** before database storage
- Exported private keys **auto-delete** after 30 seconds
- Users should delete import messages immediately
- **⚠️ This is a DEMO bot** — do not store large amounts
- Recommended: run on a private VPS, not shared hosting
